Keybinds

CTRL + ALT + H Open this page CTRL + ALT + S Open Settings CTRL + ALT + B Open new Terbium browser CTRL + ALT + C Close all windows CTRL + ALT + X Close the focused window CTRL + ALT + P Open the color picker CTRL + ALT + T Open the terminal SHIFT + Left Arrow Shifts the focused window to the left SHIFT + Right Arrow Shifts the focused window to the right SHIFT + Up Arrow Shifts the focused window SHIFT + Down Arrow Minimizes the focused window up maximized CTRL + ALT + N Minimize the focused window CTRL + ALT + Q Reload the focused window
